Mon projet final  est une destination de luxe offrant une sélection des meilleurs   
parfums de niche, le client peut visualiser les parfum dans la
page d’accueil et crée un compte pour faire les démarche des achats et en suite il 
peut laisser un message ou commentaire.

Liste approche agile :
1. Page d’accueil (html.css) 
      2. Page achats
             2.a) Créer un formulaire, 
,           2.b) Créer une base de données pour les comptes(phpmyadmin) 
            2.c) Sécurisation des mots de passe(hachage, validation expression régulière)
     3. Page contact (html.css) 
               Créer une base de données pour les achats(phpmyadmin)
